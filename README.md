## Sorting Utility

This repository also includes a small Python sorting program in `sorting.py`.

Run the demo:

```bash
python sorting.py
```

Sort your own integers:

```bash
python sorting.py 8 3 5 1 9
```

Or use standard input:

```bash
echo 8 3 5 1 9 | python sorting.py
```

Run the tests:

```bash
python -m unittest
```
