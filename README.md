# My OSS Contribution Plan

## Current Thoughts on Open Source Contribution

I currently hope to start with small but meaningful contributions to open source
projects. My first goal is to improve my ability to read unfamiliar codebases,
understand contribution guidelines, and communicate clearly through issues and
pull requests. I also want to practice fixing documentation problems, simple
bugs, and small utility scripts before trying larger features.

## Goals for This Semester

During this semester, I want to build a solid Git and GitHub workflow, learn
how to collaborate in a standard open source process, and complete at least one
real contribution with a clear commit history. I also want to become more
confident in reviewing code, tracking changes, and using branches to manage
different tasks safely.
